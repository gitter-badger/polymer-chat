    Polymer Chat

    Welcome to the demo page of Polymer Chat, a simple Polymer element to display a completely anonymous chat widget on your web site.

    Please note that this project is an experiment and as such is not meant at all to be released in the wild as-is!

    Every feedback as well as pull requests are very well accepted.

    Highlights

    • Support for ARIA Live Regions (thanks to @karlgroves for its help)
    • Connects to a Primus server hosted on Modulus
